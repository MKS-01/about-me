READBACK(1)Weekend Hacks Manual

NAME

readback — any article → offline audiobook

DESCRIPTION

Paste a URL. Reader-mode strips the junk, CSM-1B on MLX reads it back in a cloned voice — all on Apple Silicon. No cloud, no API keys; your reading list stays yours.

STACK

mlx(1), csm-1b(1), fastapi(1), bun(1), ink(1)
